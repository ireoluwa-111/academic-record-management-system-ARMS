Administrator / Roles & Permissions

System Roles

Baseline security and access groups

Admin

Complete system oversight, database control, user lifecycle management, and security settings.

  • Manage all user accounts
  • Create academic departments
  • View system audit trails
  • Configure system settings

Registrar

Handles curriculum management, enrollment oversight, grade approval, and official transcript generation.

  • Approve student registrations
  • Create and update courses
  • Generate academic transcripts
  • Search all student profiles

Lecturer

Manages assigned courses, retrieves student lists, submits student scores, and prints class lists.

  • View assigned courses
  • Access student rosters
  • Submit & edit course marks
  • View performance stats

Student

Accesses registration, enrolls in courses, views grade reports, and requests academic transcripts.

  • Register courses inline
  • View semester results
  • Request and print transcripts
  • Update profile details
